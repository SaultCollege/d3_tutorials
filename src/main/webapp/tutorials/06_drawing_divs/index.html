Drawing divs...


Draw divs, modify attributes and change styles.

Draw a div with 
<div style="display: inline-block;
    width: 20px;
    height: 75px;
    background-color: teal;">
</div>



Create a style 
div.bar {
    display: inline-block;
    width: 20px;
    height: 75px;   /* We'll override this later */
    background-color: teal;
}

Apply the style 
<div class="bar"></div>


Use D3's selection.attr() to add a class to an element
To give divs a class of bar, we can use: 
.attr("class", "bar")

Create divs and set class to "bar" 
var dataset = [ 5, 10, 15, 20, 25 ];

d3.select("body").selectAll("div")
    .data(dataset)
    .enter()
    .append("div")
    .attr("class", "bar");

Set the bars height by changing the style height
You can change style with d3.style("height", function(d))
The code 
var dataset = [ 5, 10, 15, 20, 25 ];

d3.select("body").selectAll("div")
    .data(dataset)
    .enter()
    .append("div")
    .attr("class", "bar")
    .style("height", function(d) {
    return d + "px";
});
